01 — Getting Started
Setup Instructions
Follow these steps to set up your Canva Brand Kit. You'll need Canva Pro or Canva for Teams to access Brand Kit.
1
Open Brand Kit
Canva → Brand Hub → Brand Kit (or canva.com/brand)

If you already have a Brand Kit at the URL you shared, open it and edit it. Otherwise, create a new one named "ZeroOne D·O·T·S AI".

2
Upload Logos

Upload all logo variants you have (primary light, primary dark, horizontal, icon-only, social avatar). Canva supports SVG and PNG. Organize them by clicking "Add new logo" for each variant.

Tip: Upload the SVG versions first — they scale perfectly. Add PNG versions as fallbacks. Your logo variants include: Primary Light, Primary Dark, Horizontal Light, Horizontal Dark, Icon Only, Social Avatar.
3
Add Brand Colors

Click "Add a color palette" and enter each hex code from the tables below. Create separate palettes for the two modes:

Palette 1: "Website / Editorial" (pastel on warm backgrounds)

Palette 2: "Instagram / Social" (vibrant on dark backgrounds)

Palette 3: "Neutrals & Anchors" (deep tones + warm neutrals)

4
Add Brand Fonts

Under "Brand fonts," upload or search for each font. Set the hierarchy:

• Title: Orbitron (Bold 900) — always uppercase
• Subtitle: Syne (SemiBold 600, italic) — editorial feel
• Body: Space Mono (Regular 400) — data/monospace vibe

Important: Orbitron, Syne, and Space Mono are all on Google Fonts and should be available in Canva's font library. Search for them by name. If not found, upload the .ttf/.otf files from fonts.google.com.
5
Set Brand Voice (Optional)

Under Brand Voice / Brand guidelines, add these notes:

"Contrarian, confident, data-backed. We challenge subscription AI with private infrastructure. Tone: bold but warm, technical but accessible. Tagline: Own your AI. Don't rent it. Never use: corporate jargon, passive voice, uncertain hedging."

04 — Typography
Three Voices
Each font serves a specific purpose. Set these as your brand fonts in Canva's Brand Kit typography section.
FONT 1 — TITLE / HEADING
ORBITRON
ABCDEFGHIJKLMNOPQRSTUVWXYZ 0123456789
Google Fonts Weight: 900 (Black) ALWAYS UPPERCASE Headlines · Titles · Numbers · Brand name
Canva setup: Search "Orbitron" in brand fonts → Set as "Title" → Weight: Bold/Black → Transform: UPPERCASE
FONT 2 — SUBTITLE / EDITORIAL
Syne — The editorial voice
Aa Bb Cc Dd Ee Ff Gg Hh Ii Jj Kk Ll Mm Nn Oo Pp Qq Rr Ss Tt Uu Vv Ww Xx Yy Zz
Google Fonts Weight: 600 (SemiBold) Often italic Subtitles · Taglines · Editorial feel
Canva setup: Search "Syne" → Set as "Subtitle" → Weight: SemiBold → Style: Italic for taglines
FONT 3 — BODY / DATA
Space Mono — the data layer
Aa Bb Cc Dd Ee Ff Gg Hh Ii Jj Kk Ll Mm Nn Oo Pp Qq Rr Ss Tt Uu Vv Ww Xx Yy Zz 0123456789
Google Fonts Weight: 400 (Regular) / 700 (Bold) Monospace Body · Data labels · Tags · Source citations
Canva setup: Search "Space Mono" → Set as "Body" → Weight: Regular → Size: 11-14px range
⚠️ NEVER use: Inter, Roboto, Arial, Helvetica, or any system/default fonts. These break the brand identity.
